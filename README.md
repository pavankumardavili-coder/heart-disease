# ❤️ Heart Disease Analysis and Visualization Dashboard

---

# 📌 Project Overview

The Heart Disease Analysis and Visualization Dashboard is an interactive data visualization project developed to analyze patient clinical records, cardiac risk factors, hospital infrastructure, diagnosis patterns, treatment costs, and regional heart disease case distribution.

The project integrates multiple heart disease datasets using MySQL for data management and SQL for data analysis. Interactive dashboards and stories are created using Tableau Desktop, published through Tableau Public, and embedded into a Flask web application for an intuitive and user-friendly experience.

---

# 🎯 Project Objectives

The main objectives of this project are:

- Analyze patient clinical indicators linked to heart disease.
- Compare risk factors such as cholesterol, blood pressure, and chest pain type.
- Visualize patient diagnosis distribution.
- Analyze heart rate response and exercise-induced angina patterns.
- Compare treatment cost and recovery time across risk categories.
- Study chest pain type and thalassemia test outcomes by gender.
- Provide interactive dashboards for data-driven clinical decision-making.
